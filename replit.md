# 簿記マスター - 勘定科目分類ゲーム

## 概要

簿記3級レベルの初心者向け学習アプリ。勘定科目を5要素（資産・負債・純資産・収益・費用）に分類するドラッグ＆ドロップのミニゲームです。

## 技術スタック

- **フロントエンド**: React 19 + TypeScript + Vite
- **スタイリング**: Tailwind CSS 4 + shadcn/ui
- **ドラッグ＆ドロップ**: @dnd-kit/core
- **アニメーション**: Framer Motion
- **ルーティング**: wouter
- **データ永続化**: localStorage

## プロジェクト構成

```
client/src/
├── components/game/    # ゲーム専用コンポーネント
├── context/            # React Context（GameContext）
├── data/               # 勘定科目マスタデータ
├── lib/                # ユーティリティ、ストレージ操作
├── pages/              # 各画面コンポーネント
└── App.tsx             # ルート設定

shared/
└── schema.ts           # 型定義（Zod）
```

## 画面構成

1. **ホーム** (`/`) - メインメニュー、成績表示
2. **ゲーム** (`/game`) - ドラッグ＆ドロップ分類ゲーム
3. **結果** (`/result`) - スコア、称号、間違い一覧
4. **弱点帳** (`/weakpoints`) - 苦手科目の復習
5. **設定** (`/settings`) - 難易度、カード枚数、制限時間

## 状態管理

`GameContext`でゲーム全体の状態を管理:
- ユーザーデータ（成績、設定、弱点）
- ゲームフェーズ（idle/playing/result）
- 現在のカード、スコア、コンボ

## データ永続化

localStorageを使用してブラウザに保存:
- キー: `boki_game_data`
- 内容: 成績履歴、弱点科目、設定

## ユーザー設定

- 難易度: 初級（12科目）/ 中級（30科目）
- カード枚数: 5〜20枚
- 制限時間: オフ / 30〜120秒
