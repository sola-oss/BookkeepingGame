# 簿記マスター - 勘定科目分類＆仕訳ゲーム

## 概要

簿記3級レベルの初心者向け学習アプリ。6つのモードで簿記の基礎を学べます：
- **分類モード**: 勘定科目を6要素（資産・負債・純資産・収益・原価・経費）に分類（ドラッグ＆ドロップ）
- **仕訳モード**: 取引文から借方・貸方を完成させる（選択式）
- **試験モード**: 記述式で勘定科目名・金額を入力（同義語辞書によるファジーマッチング対応）
- **模試モード**: 日商簿記3級形式の3大問構成（仕訳・勘定記入・決算）、60分タイマー
- **教科書モード**: 初学者向け解説ページ（一言まとめ、取引パターン、間違えがちポイント、ミニ問題）
- **会計フローモード**: 仕訳から決算書への流れを可視化（アニメーション付き）

## 技術スタック

- **フロントエンド**: React 19 + TypeScript + Vite
- **スタイリング**: Tailwind CSS 4 + shadcn/ui
- **ドラッグ＆ドロップ**: @dnd-kit/core
- **アニメーション**: Framer Motion
- **ルーティング**: wouter
- **データ永続化**: localStorage

## プロジェクト構成

```
client/src/
├── components/game/    # ゲーム専用コンポーネント
├── context/            # React Context（GameContext, JournalContext, ExamContext, AccountingFlowContext）
├── data/               # 勘定科目マスタデータ（50科目）、仕訳問題（15問）、試験問題（40問）、教科書（10トピック）
├── lib/                # ユーティリティ、ストレージ操作
├── pages/              # 各画面コンポーネント
└── App.tsx             # ルート設定

shared/
└── schema.ts           # 型定義（Zod）、バッジ定義
```

## 画面構成

1. **ホーム** (`/`) - メインメニュー（3モード選択）、成績表示
2. **分類ゲーム** (`/game`) - ドラッグ＆ドロップ分類ゲーム
3. **分類結果** (`/result`) - スコア、称号、間違い一覧、仕訳例表示
4. **仕訳ゲーム** (`/journal`) - 借方・貸方を選択する仕訳ゲーム
5. **仕訳結果** (`/journal-result`) - 仕訳モードの結果表示
6. **試験開始** (`/exam-start`) - 試験モードの設定（問題数・制限時間）
7. **試験モード** (`/exam`) - 記述式入力、タイマー、ナビゲーション
8. **試験確認** (`/exam-review`) - 回答一覧、フラグ付き問題確認
9. **試験結果** (`/exam-result`) - 採点結果、正誤詳細表示
10. **弱点帳** (`/weakpoints`) - 苦手科目の復習
11. **設定** (`/settings`) - 難易度（初級/中級/上級）、カード枚数、制限時間
12. **バッジ** (`/badges`) - 獲得バッジコレクション
13. **財務諸表** (`/statements`) - 貸借対照表・損益計算書のプレビュー
14. **勘定科目一覧** (`/accounts`) - 科目辞書、検索・フィルター・モーダル詳細
15. **教科書** (`/textbook`) - 10トピックの学習ページ（検索・モーダル詳細）
16. **模試開始** (`/mock-exam-start`) - 模試モードの説明・開始
17. **模試** (`/mock-exam`) - 3大問構成の模試、60分タイマー
18. **模試結果** (`/mock-exam-result`) - セクション別採点結果
19. **会計フロー** (`/accounting-flow`) - 仕訳→決算書の流れを可視化

## 状態管理

**GameContext** - 分類モードの状態管理:
- ユーザーデータ（成績、設定、弱点）
- ゲームフェーズ（idle/playing/result）
- 現在のカード、スコア、コンボ
- フィードバック表示状態

**JournalContext** - 仕訳モードの状態管理:
- 仕訳問題リスト、現在の問題インデックス
- 借方・貸方の選択状態
- スコア、コンボ、フィードバック

**ExamContext** - 試験モードの状態管理:
- 試験フェーズ（idle/taking/reviewing/result）
- 問題リスト、回答状態、フラグ管理
- タイマー（カウントダウン）、自動提出
- 採点結果（GradingResult配列）

**AccountingFlowContext** - 会計フローモードの状態管理:
- フェーズ（idle/answering/animating/complete）
- 5問のサンプル仕訳問題（購入・販売・借入・給料・回収）
- P/L・B/S集計トータル（費用・収益・資産・負債・純資産）
- アニメーション状態（チップ移動、純利益振替）
- prefers-reduced-motion対応

## データ永続化

localStorageを使用してブラウザに保存:
- キー: `boki_game_data`
- 内容: 成績履歴、弱点科目、設定、連続日数

## ユーザー設定

- 難易度: 初級（12科目）/ 中級（30科目）/ 上級（50科目）
- カード枚数: 5〜20枚
- 制限時間: オフ / 30〜120秒

## バッジシステム

10種類の達成バッジ:
- 初めの一歩、練習家、学習熱心、簿記マスター（ゲーム回数）
- 3日連続、週間チャンピオン（連続日数）
- 正解50、正解100、正解王（累計正解数）
- パーフェクト（100%正答率）

## 教科書モード

10トピックの教科書ページ:
1. 売掛金・買掛金（信用取引）
2. 仕入・売上（三分法）
3. 立替金・預り金
4. 前払費用・前受収益
5. 未払費用・未収収益
6. 棚卸（期末商品・しーくりくりしー）
7. 現金・預金
8. 給料など主要費用
9. 借入金と返済
10. 試算表の借方・貸方の増減ルール

各ページの構成:
- 一言まとめ（summary）
- よくある取引パターン（transactionPatterns）
- 間違えがちなポイント（commonMistakes）
- 見分けチェック（checklist）
- ミニ問題（miniQuiz）

試験結果との連携:
- 試験問題に `topic_tag` フィールドを追加
- 結果画面から該当する教科書ページへ「復習」導線を表示
