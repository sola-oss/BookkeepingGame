# 簿記マスター - 勘定科目分類＆仕訳ゲーム

## 概要

簿記3級レベルの初心者向け学習アプリ。2つのゲームモードで簿記の基礎を学べます：
- **分類モード**: 勘定科目を5要素（資産・負債・純資産・収益・費用）に分類
- **仕訳モード**: 取引文から借方・貸方を完成させる

## 技術スタック

- **フロントエンド**: React 19 + TypeScript + Vite
- **スタイリング**: Tailwind CSS 4 + shadcn/ui
- **ドラッグ＆ドロップ**: @dnd-kit/core
- **アニメーション**: Framer Motion
- **ルーティング**: wouter
- **データ永続化**: localStorage

## プロジェクト構成

```
client/src/
├── components/game/    # ゲーム専用コンポーネント
├── context/            # React Context（GameContext, JournalContext）
├── data/               # 勘定科目マスタデータ（50科目）、仕訳問題（15問）
├── lib/                # ユーティリティ、ストレージ操作
├── pages/              # 各画面コンポーネント
└── App.tsx             # ルート設定

shared/
└── schema.ts           # 型定義（Zod）、バッジ定義
```

## 画面構成

1. **ホーム** (`/`) - メインメニュー（分類・仕訳モード選択）、成績表示
2. **分類ゲーム** (`/game`) - ドラッグ＆ドロップ分類ゲーム
3. **分類結果** (`/result`) - スコア、称号、間違い一覧、仕訳例表示
4. **仕訳ゲーム** (`/journal`) - 借方・貸方を選択する仕訳ゲーム
5. **仕訳結果** (`/journal-result`) - 仕訳モードの結果表示
6. **弱点帳** (`/weakpoints`) - 苦手科目の復習
7. **設定** (`/settings`) - 難易度（初級/中級/上級）、カード枚数、制限時間
8. **バッジ** (`/badges`) - 獲得バッジコレクション
9. **財務諸表** (`/statements`) - 貸借対照表・損益計算書のプレビュー

## 状態管理

**GameContext** - 分類モードの状態管理:
- ユーザーデータ（成績、設定、弱点）
- ゲームフェーズ（idle/playing/result）
- 現在のカード、スコア、コンボ
- フィードバック表示状態

**JournalContext** - 仕訳モードの状態管理:
- 仕訳問題リスト、現在の問題インデックス
- 借方・貸方の選択状態
- スコア、コンボ、フィードバック

## データ永続化

localStorageを使用してブラウザに保存:
- キー: `boki_game_data`
- 内容: 成績履歴、弱点科目、設定、連続日数

## ユーザー設定

- 難易度: 初級（12科目）/ 中級（30科目）/ 上級（50科目）
- カード枚数: 5〜20枚
- 制限時間: オフ / 30〜120秒

## バッジシステム

10種類の達成バッジ:
- 初めの一歩、練習家、学習熱心、簿記マスター（ゲーム回数）
- 3日連続、週間チャンピオン（連続日数）
- 正解50、正解100、正解王（累計正解数）
- パーフェクト（100%正答率）
